# Release v0.1.3

## Overview

v0.1.3 adds SSH access to the Raspberry Pi 5 BusyBox initramfs image using Dropbear and public-key authentication.

## Features

- BusyBox-based initramfs boot
- Ethernet bring-up during boot
- DHCP client integration using `udhcpc`
- Static IP fallback when DHCP fails
- Dropbear SSH server integration
- Public-key authentication for root login
- Persistent SSH host key packaging into the initramfs
- `/etc/shells` support for BusyBox ash login
- Startup permission fix for `/root/.ssh/authorized_keys`

## Validation

- Verified Dropbear server startup
- Verified network link and IP assignment
- Verified SSH login from host laptop
- Verified public-key authentication
- Verified root shell validation through `/etc/shells`

## Notes

This release is still initramfs-based. Host keys and authorized keys are packaged into `rootfs.cpio.gz` so the system is ready at boot time.
